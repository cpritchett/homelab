# Constitutional Amendments

Amendments to the constitution are recorded here. Each amendment must:

1. Reference the principle being modified (or establish new procedures)
2. Explain the rationale for the change
3. Be approved through the standard ADR process

## Format

Use `AMENDMENT-NNNN-short-title.md` naming convention.
