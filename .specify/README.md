# .specify

This folder enables Spec-Kit compatibility for specification-driven development.

## Structure

- `memory/` — Persistent context (constitution link)
- `templates/` — Reusable templates for specs, plans, tasks

## Usage

Spec-Kit slash commands:
- `/speckit.constitution` — Review project principles
- `/speckit.specify` — Define requirements for a feature
- `/speckit.plan` — Choose implementation approach
- `/speckit.tasks` — Generate ordered tasks
- `/speckit.implement` — Execute tasks
